# Instagram Posts Dataset Sample

<h2>7,730件のレコードを含むサンプルdataset</h2>

<a href="https://brightdata.jp">
    <img src="https://github.com/bright-jp/Instagram-Posts-dataset-samples/blob/main/datasets-image.png" alt="Bright Data datasets" />

この**Instagram Posts Dataset Sample**には**7,730件のレコード**が含まれており、**Bright Data API**を使用して抽出されています。

## データポイント

| フィールド名 | 説明 |
|------------|-------------|
| `url` | Instagram投稿への直接リンク。 |
| `user_posted` | コンテンツを投稿したアカウントのユーザー名。 |
| `description` | Instagram投稿に付随するキャプションテキスト。 |
| `hashtags` | 投稿で使用されたハッシュタグの一覧。コンテンツ発見に役立ちます。 |
| `num_comments` | 投稿のコメント総数。 |
| `date_posted` | 投稿が公開されたタイムスタンプ。 |
| `likes` | 投稿が受け取った「いいね」の数。 |
| `photos` | 投稿に含まれる写真のURL。 |
| `videos` | 投稿に含まれる動画のURL。 |
| `location` | 投稿にタグ付けされた地理的位置情報。 |
| `latest_comments` | 投稿に対する最近のコメント。 |
| `post_id` | Instagram投稿の一意の識別子。 |
| `discovery_input` | 投稿を発見するために使用されたキーワードまたは入力。 |
| `has_handshake` | 投稿がブランドコンテンツ提携の一部であるかどうかを示します。 |
| `shortcode` | 投稿URL用の短縮された一意コード。 |
| `content_type` | 投稿の形式。例：写真、動画、またはカルーセル。 |
| `pk` | 内部データ処理用の主キー識別子。 |
| `content_id` | コンテンツ用にシステム生成されたID。 |
| `engagement_score_view` | 投稿のエンゲージメントを表す計算済みメトリクス。 |
| `thumbnail` | 投稿のサムネイル画像のURL。 |
| `video_view_count` | 動画が視聴された回数。 |
| `product_type` | 投稿の分類。例：標準コンテンツまたはブランドコンテンツ。 |
| `coauthor_producers` | 投稿に関連する共同作成者またはプロデューサーのユーザー名。 |
| `tagged_users` | 投稿でタグ付けされたユーザーの一覧。 |
| `video_play_count` | 動画が再生された回数。 |
| `followers` | 投稿アカウントのフォロワー数。 |
| `posts_count` | ユーザーが行った投稿の総数。 |
| `profile_image_link` | ユーザーのプロフィール画像のURL。 |
| `is_verified` | ユーザーのアカウントがInstagramによって認証されているかどうかを示します。 |
| `is_paid_partnership` | 投稿が有料パートナーシップであるかどうかを示します。 |
| `partnership_details` | パートナーシップまたはコラボレーションの詳細。 |
| `user_posted_id` | コンテンツを投稿したユーザーの一意の識別子。 |
| `post_content` | テキストやメディアを含む投稿のメインコンテンツ。 |
| `audio` | 投稿で使用された音声の詳細。 |
| `profile_url` | ユーザーのInstagramプロフィールへのURL。 |
| `videos_duration` | 投稿内の動画コンテンツの長さ。 |
| `images` | 投稿に関連する画像の配列。 |
| `alt_text` | 投稿画像の代替テキスト説明。 |
| `photos_number` | 投稿に含まれる写真の数。 |
| `audio_url` | 投稿で使用された音声へのリンク。 |


## ユースケース

### 1. ソーシャルメディアトレンド分析
投稿のエンゲージメント指標を分析して、Instagram上でトレンドとなっているコンテンツやユーザーの嗜好を特定します。

### 2. インフルエンサーマーケティング戦略
ユーザーおよびエンゲージメントデータを活用して、ブランドコラボレーションに最も効果的なインフルエンサーを選定します。

### 3. 競合分析
競合他社のエンゲージメントやコンテンツ戦略を監視し、自社のソーシャルメディアアプローチの改善に役立てます。
